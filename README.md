## Project Overview
This project is an interactive analytics dashboard built to evaluate multi-channel marketing performance. It processes over **4.9 million conversions** to provide actionable insights into ROI and campaign success.

## Key Features
- **A/B Testing**: Compared conversion rates across different segments.
- **Machine Learning**: Utilized a `RandomForestClassifier` to predict campaign success probability.
- **Attribution Modeling**: Implemented logic to compare Last-Click vs. Linear attribution for budget optimization.
- **Real-time ROI Tracking**: Visualized cost-effectiveness with an average conversion rate of **60.98%**.

## Tech Stack
- Python: Core logic and data engineering.
- Pandas: Data manipulation of large-scale marketing datasets.
- Streamlit: Web-based interactive UI and deployment.
- Scikit-learn: Predictive modeling and performance evaluation.
